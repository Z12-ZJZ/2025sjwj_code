# 数据挖掘导论课程作业

本项目包含五个数据挖掘任务的完整实现，涵盖聚类、异常检测、时间序列预测等多个领域。

## 项目结构

```
2025sjwj_code/
├── 任务 1：聚类任务/              # 图像聚类任务
├── 任务 2：图像异常检测任务/       # 图像异常检测
├── 任务 3：时间序列预测任务/     # 天气温度预测
├── 任务 4：无监督疾病判断任务/     # 甲状腺异常检测
├── 任务 5：多模态异常检测统一框架分析/  # 统一框架设计与分析
└── README.md
```

## 任务概览

### 任务 1：聚类任务
- **目标**：对600张图像进行聚类分析，识别6个类别（cable, tile, bottle, pill, leather, transistor）
- **方法**：多种特征提取方法（颜色直方图、像素特征、组合特征）+ K-means/层次聚类
- **评估**：轮廓系数、调整兰德指数、聚类纯度、标准化互信息
- **位置**：`任务 1：聚类任务/Cluster/`

### 任务 2：图像异常检测任务
- **目标**：使用无监督方法检测图像中的异常区域（hazelnut和zipper两类）
- **方法**：基于Wide ResNet50-2的深度特征提取 + 多元高斯分布建模 + 马氏距离
- **特点**：可生成异常热力图，实现像素级异常定位
- **位置**：`任务 2：图像异常检测任务/Image_Anomaly_Detection/`

### 任务 3：时间序列预测任务
- **目标**：预测室外温度（OT）时间序列
- **方法**：LSTM/GRU循环神经网络 + 滑动窗口 + 特征工程
- **特点**：支持多种特征工程选项（时间特征、特征选择等）
- **位置**：`任务 3：时间序列预测任务/weather_prediction/`

### 任务 4：无监督疾病判断任务
- **目标**：基于6个特征检测甲状腺异常（仅使用正常样本训练）
- **方法**：Isolation Forest无监督异常检测
- **评估**：ROC-AUC、PR-AUC、混淆矩阵等
- **位置**：`任务 4：无监督疾病判断任务/thryroid/`

### 任务 5：多模态异常检测统一框架分析
- **目标**：分析任务2（图像）和任务4（表格）的异同，设计统一检测框架
- **方法**：分层统一框架设计（统一接口 + 模态特化实现）
- **特点**：自动模态识别、可扩展架构
- **位置**：`任务 5：多模态异常检测统一框架分析/`
